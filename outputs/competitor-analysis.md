## IMPORTANT NOTE: Microsoft Math Solver

Microsoft Math Solver (ID: 1217540251) has been **discontinued**. The mobile apps were removed from the App Store in August 2024, and the web service was fully retired on July 7, 2025. Math functionality has been absorbed into OneNote's Math Assistant.

**Replacement:** Gauth: AI Study Companion (1.37M ratings) is now the #2 most-rated math app and has been included as the 5th competitor instead.

Sources:
- [Microsoft Math Solver retirement announcement](https://x.com/MicrosoftMath/status/1931103750753599946)

## GAPS AND OPPORTUNITIES FOR MATHPRO

### GAP 1: "Homework Helper" is underused
- Only Mathway uses "homework" in subtitle
- "Homework helper" is a massive search term for students
- **Opportunity:** Include "homework" in subtitle or keyword field
- **MathPro Action:** Consider subtitle like "Math Homework Helper & Solver"

### GAP 2: No competitor combines "AI" + "step-by-step" in title/subtitle
- Symbolab uses "AI" in title but not "step-by-step"
- Photomath uses "step-by-step" in subtitle but not "AI"
- **Opportunity:** MathPro can own the "AI step-by-step" combination
- **MathPro Action:** Title: "MathPro: AI Math Problem Solver" / Subtitle: "Step-by-Step Homework Helper"

### GAP 3: No competitor mentions Apple Pencil or iPad optimization
- Zero competitors reference Apple Pencil input in any metadata
- Zero competitors emphasize iPad experience
- **Opportunity:** Unique differentiator for MathPro
- **MathPro Action:** Include "Apple Pencil" in description, show iPad screenshots

### GAP 4: No competitor mentions practice problems or quizzes in title/subtitle
- Gauth mentions quizzes in description but not in title/subtitle
- No competitor positions as "solve + practice" in visible metadata
- **Opportunity:** "Practice" and "quiz" keywords are uncontested in titles
- **MathPro Action:** Could test subtitle: "Solve, Learn & Practice Math"

### GAP 5: No competitor uses "tutor" in title or subtitle
- Gauth has "AI Live Tutor" as a feature but not in title/subtitle
- "Math tutor" is a high-volume search term
- **Opportunity:** Claim "tutor" keyword in subtitle or keyword field
- **MathPro Action:** Keyword field should include "tutor, math tutor, AI tutor"

### GAP 6: "Offline" is completely absent
- No competitor mentions offline capability anywhere in title/subtitle
- Students need math help without internet (exams, travel, poor connectivity)
- **Opportunity:** If MathPro has any offline mode, this is unique
- **MathPro Action:** If feasible, mention "offline" in description

### GAP 7: Color branding opportunity
- Photomath: Pink/Yellow/Blue (playful)
- Mathway: Navy Blue (corporate)
- Symbolab: Red + Graph paper (school)
- Question.AI: Bright Blue (tech)
- Gauth: Purple/Blue gradients
- **Opportunity:** GREEN and ORANGE are completely unoccupied
- **MathPro Action:** Use green or orange as primary brand color to stand out visually in search results

### GAP 8: "Family" or "Classroom" keywords absent
- No competitor targets parents or teachers in title/subtitle
- "Family plan" and "classroom" are uncontested
- **Opportunity:** If MathPro offers family/classroom features, own this space
- **MathPro Action:** Consider "Family Math Helper" messaging in description

### GAP 9: Weak descriptions
- Mathway's description is almost entirely subscription legal text
- Question.AI's description focuses on general AI chatbot, not math
- Symbolab's description is feature-list heavy but lacks emotional hooks
- **Opportunity:** MathPro can win with a well-crafted, keyword-rich, emotionally engaging description
- **MathPro Action:** Write a 2,200+ character description with clear sections, bullet points, and keyword density

### GAP 10: Screenshot differentiation
- Most competitors use blue/purple color schemes
- Question.AI screenshots show ZERO math content (major mismatch)
- Only Symbolab uses a unique visual theme (graph paper)
- **Opportunity:** MathPro can use a distinctive green/orange theme with math-focused screenshots
- **MathPro Action:** Every screenshot should show actual math being solved, with benefit-driven text overlays

1. **Photomath** (Strongest) -- Google ownership, Editors' Choice, 733K ratings, brand recognition
2. **Gauth** (Rising fast) -- 1.37M ratings, expanding beyond math, strong AI positioning

### Where MathPro fits:
MathPro should target the space between Symbolab (too niche, weak ratings) and Question.AI (confused positioning). By combining:
- Strong math-specific focus (unlike Question.AI and Gauth)
- AI branding (unlike Photomath and Mathway)
- Unique features (Apple Pencil, practice mode, offline)
- Distinctive visual brand (green/orange)

MathPro can carve out a position as "the modern AI math solver that actually helps you learn."
